ini format SCSS angular

> install node_module:
    npm install
> buat file angular baru:
    ng n <nama-apps>
> generate component:
    ng g c <nama-component> || ng g c <nama-file(dibuat atau sudah exist)>/<nama-component>
> generate module:
    ng g m <nama-module>
> generate service:
    ng f s <nama-service>
> Add Angular Material Design:
    ng add @angular/material
    (=> Y, => Y)

> download UIKit
    https://getuikit.com/docs/introduction
    => import di style.scss, like: @import './uikit/css/uikit.css';

>   git merge
    see what branch u used with $git branch, then ke branch yg ingin dimerge with $git checkout <nama-branch>,
    pull first with $git pull origin <nama-branch>, then $git merge <nama-branch-asal-yang ingin di merge>, maybe some codes are conflict so watch carefully the codes what u want accept or not,
    next see the changes with $git status or $git diff or $alias log="git log --all --decorate --oneline --graph" to see with $git log,
    make sure and then $git add ., commit with $git commit -m "<comment-commit>",
    last push with $git push origin <nama-branch>.

> update personal token access token on github
    on github, profile nav -> settings -> developer settings -> Personal access tokens -> generate new token -> copied.
    on windows, search credential manager -> windows credentials -> find git:https://github -> change the password with the token.
    on git bash, command git push origin <name-branch> -> in usernam enter the username github, in password enter the token.
